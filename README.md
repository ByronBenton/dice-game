# Dice Game

A simple interactive dice game built with HTML, CSS, and JavaScript. Users can roll two dice and see which one wins.

**Purpose:**  
Built as a small experiment to explore interactive UI design and user feedback loops.

**Role / Decisions:**  
- Designed the user interface and layout  
- Implemented game logic in JavaScript  
- Iterated on UX to make the dice rolls clear and engaging

**Learnings / Next Steps:**  
- Learned about event handling, DOM manipulation, and basic game mechanics  
- Could expand with scoring, multiplayer, or animations for enhanced engagement
